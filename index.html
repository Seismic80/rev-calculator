Mechanics & notes

  • Weapon/unique drop rates are 5× higher on a Krystilia revenants task; the boost does not apply to a ghosts task. Revenants
  • When the unique table is rolled: 2/5 amulet of avarice, 1/5 each weapon. Unique table chance is 1/(A×26.667) unskulled off task, where A = floor(2200 / floor(√combat)); on task ×5 better, skulled ×1/0.55 (≈45%) better, and the two stack. Template:Revenants/Drops
  • Off-task rates are computed as exactly on-task ÷ 5. The wiki footnotes round the multipliers independently (26.667 vs 5×5.333), a <0.01% difference.
  • Expected amulets of avarice and non-target weapons obtained en route are identical on- and off-task (5× rate exactly offsets 5× kills), so they contribute 0 GP to the loot difference.
  • Ancient artefact / seed / dragon med helm rates don't depend on task status. Skulled shifts that table (most artefacts ≈1.8× more common, ancient emblem rarer, dragon med helm unobtainable). source
  • Coins are dropped only by revenants below combat 132 (chance (A−B)/A per the wiki drop lines, B = 15 + floor((combat+60)²/200)); revenant dragons drop none.
  • Krystilia revenant tasks: 40–100 kills (avg 70), or 100–150 (avg 125) with the Revenenhanced extension. Task weight 5/196. Krystilia
  • Revenants give Slayer XP ≠ max HP: it's hp × (1 + species XP bonus) per each species' wiki infobox — e.g. dragon 155 hp but 186 XP, hobgoblin 72 hp / 73 XP, imp 10/10.
  • Turael filler XP assumes one completed Turael task per non-rev Krystilia roll (Turael can only replace a task once, so fillers must be finished before re-rolling). Per-task kill counts and weights from Turael's assignment list; XP/kill uses the variant at the wiki's suggested farming spot. Blocks are account-wide, so a blocked monster is also never assigned by Turael.
  • Larran's keys drop only on a Krystilia task: 1/floor(99 − (combat−81) × 49/269) per kill at combat 81+ (1/89 at dragons), 1/(100 + floor(0.3×(80−combat)²)) below. source. Key counts are shown but their chest GP isn't added to the loot comparison; the on-task-only Slayer's enchantment drops also aren't counted.
  • Tradeable drops are valued at max(GE mid-price, high alch), the wiki's convention. Ancient artefacts use GE price (Emblem Trader pays slightly more: 500k–16m).
